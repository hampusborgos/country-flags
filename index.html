All country flags in SVG

Accurate renderings of the flags of all countries available as PNG and SVG.

Easiest way to get them is to clone the git repository:

git clone git@github.com:hampusborgos/country-flags.git

Or go to the Github page to read more

Kosovo is listed as XK. This is a user-assigned code per ISO-3166, and is in use by several multinational organizations.
